Complaint Locations

This classwork challenges students to display static maps for locations of complaints made through 311 Service Request calls from NYC Open Data. Map functionality is provided through LeafletJS. Students will practice using information retrieved from one RESTful API call as the parameters to another JS library. The latitude and longitude information of a complaint retrieved from NYC Open Data will be used with LeafletJS to create a map of the complaint.
